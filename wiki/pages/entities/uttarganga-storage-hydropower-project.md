# Uttarganga Storage Hydropower Project

Uttarganga is an 828 MW storage hydropower project in Baglung promoted by NEA. It is pinned in the [[data-layer-top-10-capacity-projects]] layer because it is the fourth-largest capacity marker in the display set and one of the few top-10 records that is explicitly a storage project.

## Why It Matters

Uttarganga matters because it combines two properties that rarely overlap in Nepal's project stack: very large capacity and storage. Most top-10 capacity markers are survey-stage run-of-river or peaking run-of-river concepts. Uttarganga is a direct part of the [[storage-deficit]] question.

Its location in Gandaki Province also makes it a counterweight to the Karnali-heavy top-10 list. If it advanced, it would strengthen the western-central storage portfolio rather than adding another remote Karnali mainstem marker.

## Status And Timing

The current map layer treats Uttarganga as survey / study stage. NEA annual-book material provides official visual references for the project site and proposed reservoir area, but the wiki evidence set does not confirm physical construction.

The IPSDP milestone ladder groups **Uttarganga STO** in the **2031-2035** project window. That is the best planning-window anchor currently in the wiki.
