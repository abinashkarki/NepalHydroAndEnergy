# Nepal–India Electricity Transmission and Trade Project (NIETTP)

<figure class="wiki-inline-figure">
  <img src="../assets/images/niettp/nea2425-p042-img00.png" alt="400 kV Tower at Koshi River">
  <figcaption>400 kV Tower at Koshi River</figcaption>
</figure>

The World Bank-backed program that bundled Nepal's first large
cross-border-capacity push with domestic reinforcement works. In the annual
book's 2020 framing, NIETTP is the umbrella program linking the
cross-border-exchange objective to both [[hetauda-dhalkebar-inaruwa-backbone]]
and the [[hetauda-bharatpur-bardaghat-corridor]] reinforcement works.

> [!cite] nea-transmission-annual-book-2077
> Book page 66 describes NIETTP as a World Bank-funded project intended to
> establish roughly **1,000 MW** of Nepal–India high-voltage exchange
> capacity while also executing the Hetauda–Bharatpur–Bardaghat 220 kV line
> and associated substations.

## Key Facts

| Parameter | Value |
|-----------|-------|
| Program type | Transmission and trade program |
| Core objective | Increase Nepal–India exchange capacity and improve reliability |
| Funding | World Bank |
| Linked assets in 2020 book | HDDI 400 kV works; Hetauda–Bharatpur–Bardaghat 220 kV works |
| Framing | Cross-border trade + domestic reinforcement in one package |

## Why It Matters

- NIETTP is the programmatic bridge between domestic grid hardening and the
  export-opening logic that later runs through [[dhalkebar-muzaffarpur]] and
  [[india-energy-relationship]].
- It shows that Nepal's cross-border story was never just a border-wire story.
  The domestic substations and 220/400 kV reinforcements were part of the same
  reliability package.
