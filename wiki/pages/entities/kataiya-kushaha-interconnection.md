# Kataiya–Kushaha 132 kV Interconnection

The oldest Nepal–India interconnection still in operation — the
historic first link that defined the Nepal–India power trade template.
Originally energised in 1971 at 33 kV, upgraded to 132 kV, it has
been through five decades of bilateral tariff and exchange-rate
politics.

## Key Facts

| Parameter | Value |
|-----------|-------|
| Voltage | 132 kV (originally 33 kV) |
| Status | Operational (since 1971) |
| Nepal terminal | Kushaha (Sunsari) |
| India terminal | Kataiya (Bihar) |
| Historic role | First Nepal–India grid tie |
| Direction | Predominantly **imports** into eastern Nepal |
| Interconnection ID | `kataiya_kushaha` |

## Significance

- **Origin point of the Nepal–India grid relationship.** Every later
  interconnection inherits contractual and technical conventions
  established here.
- **Eastern Terai import backbone.** Still the principal import route
  for dry-season supply to Saptari–Siraha–Sunsari, where domestic
  eastern generation is thin.
- **Legacy accounting reference.** Many of the bilateral power trade
  arrangements (e.g. the Radhika energy exchange protocol) were
  framed around this single link.

## Limitations & Open Questions

- **Legacy voltage.** 132 kV thermal capacity is small relative to the
  region's modern load. During load growth spikes it has required
  load shedding or emergency procurement.
- **Planned upgrade path unclear.** Discussions to convert or parallel
  with a 220 kV link have not progressed beyond feasibility notes.
