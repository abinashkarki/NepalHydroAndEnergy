# Layer: Hydropower - Operating

**Explorer label:** Hydro — operating  
**Layer group:** Hydropower layers  
**Feature count:** 81

Operating hydropower projects from the project registry. Marker area scales by capacity, and wiki counts show page coverage.

## What It Represents

This layer shows projects that are in operation in the display registry. It is the closest map layer to Nepal's current hydropower asset base, separating built supply from future promises.

## How To Read It

Use it to understand the geography of existing generation. Marker size scales with capacity, so large stations stand out visually, but the map should be read together with technology type, river basin, seasonality, and transmission access.

## Caveats

Operating MW is not the same as firm winter power or delivered value. Run-of-river output can be highly seasonal, and a project can be technically operating while still facing evacuation or hydrology constraints.
