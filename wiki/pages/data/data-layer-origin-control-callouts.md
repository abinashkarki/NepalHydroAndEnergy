# Layer: Origin / Control Callouts

**Explorer label:** River influence  
**Layer group:** Hydrology and basin layers  
**Feature count:** 8

Interpretive callouts for origin, control class, Nepal share, and downstream identity across the basin stack.

## What It Represents

This layer provides short interpretive labels for the hydrology map. It explains whether a system is Nepal-linked, shared, or outside Nepal-origin control, and it ties that classification to downstream names and basin shares.

## How To Read It

Use callouts when presenting the map to someone who does not already know the river systems. They are designed to make the control logic visible: where Nepal is upstream, where it is only partly upstream, and where it is not upstream at all.

## Caveats

Control class is analytical shorthand. It should not be read as legal authority, treaty entitlement, or operational control over downstream water infrastructure.
