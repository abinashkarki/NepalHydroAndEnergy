# Layer: Map Text Labels

**Explorer label:** River name labels  
**Layer group:** Text label overlays  
**Feature count:** text overlay

Permanent river and basin text labels. Turn these on when geography matters more than raw features.

## What It Represents

This overlay adds permanent text labels for rivers and basins. It is a readability layer: it helps the user keep the geography in view while inspecting features that may otherwise be visually abstract.

## How To Read It

Use it during presentations, screenshots, or when comparing basins and rivers without constantly opening popups. It is especially useful when the map is zoomed to regional scale.

## Caveats

Labels can clutter dense views and may overlap at some zoom levels. Turn them off when project markers or linework need to be inspected closely.
