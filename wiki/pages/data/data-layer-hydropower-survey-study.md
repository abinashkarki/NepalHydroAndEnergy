# Layer: Hydropower - Survey/Study

**Explorer label:** Hydropower - Survey/study  
**Layer group:** Hydropower layers  
**Feature count:** 311

Survey-stage hydropower licenses: early pipeline signals, not built capacity.

## What It Represents

This layer maps projects still at survey or study stage. It is useful for understanding where developers see resource potential, but it sits far upstream from actual generation.

## How To Read It

Use it for option density, basin crowding, and long-run project geography. It is especially helpful when comparing potential-heavy regions with the actual operating fleet.

## Caveats

Survey-stage projects should not be counted as supply. Many will change size, ownership, design, timeline, or disappear entirely. This layer can easily exaggerate Nepal's practical build pipeline if read without status discipline.
